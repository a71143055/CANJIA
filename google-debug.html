📋 설정 가이드

구글 OAuth 설정이 필요하면:

  1. Google Cloud Console 방문
  2. 프로젝트 생성 또는 선택
  3. OAuth 2.0 동의 화면 설정
  4. 사용자 인증 정보 > OAuth 2.0 클라이언트 ID 생성 (웹 애플리케이션)
  5. 승인된 리디렉션 URI 추가: http://localhost:5000/google-callback.html
  6. 생성된 Client ID를 아래에 입력
