마이크로소프트 설정 확인

마이크로소프트 SDK 로딩 상태

단계별 설정 가이드

  1. Azure Portal 방문: https://portal.azure.com

    Microsoft 계정 (Outlook, Live 등)으로 로그인

  2. Azure Active Directory 선택

    좌측 사이드바에서 찾거나 상단 검색창 사용

  3. "앱 등록" 선택

    좌측 메뉴의 "앱 등록" 클릭

  4. "새 등록" 버튼 클릭

    우측 상단의 녹색 "새 등록" 버튼

  5. 앱 정보 입력
    이름: CANJIA 지원 계정 유형: 모든 조직 디렉토리의 계정 및 개인 Microsoft 계정 리디렉션 URI (웹): http://localhost:5000/ms-callback.html
  6. "등록" 버튼 클릭

    앱이 생성되면 다음 페이지로 이동

  7. Application (Client) ID 복사

    개요 페이지에서 "Application (Client) ID" 값을 복사

  8. ms-config.js 파일 수정
    clientId: "복사한 ID를 여기에 붙여넣기"

⚠️ 일반적인 문제

Q: "Invalid redirect_uri" 오류

Azure Portal에 등록된 리디렉션 URI가 ms-config.js의 redirectUri와 정확히 일치하는지 확인하세요.
http://localhost:5000/ms-callback.html

Q: 로그인 버튼이 작동하지 않음

1. ms-config.js에 Client ID가 입력되었는지 확인
2. 마이크로소프트 SDK가 로드되었는지 확인 (아래 상태 확인)
3. 개발자 도구(F12)의 콘솔에서 오류 메시지 확인

Q: 프로필이 저장되지 않음

1. Flask 백엔드 서버가 실행 중인지 확인: python app.py
2. 개발자 도구 Network 탭에서 /api/profiles 요청 확인
3. 백엔드 콘솔에서 오류 메시지 확인
